## Overview

A tmux list view tool designed to monitor and display agent statuses, with integrations for kiro CLI and Claude Code.

## Features

- Real-time agent status monitoring in tmux
- Integration with kiro CLI
- Integration with Claude Code
- List view for easy status tracking

## Technical Details

Built to provide visibility into agent operations and statuses directly within the tmux environment.
